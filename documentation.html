Froblox : La Documentation



[ Le contenu du bot ]


Actuellement, on utile les slash commands donc : /.
Par ailleurs, vous avez des choix disponibles! Vous avez deux commandes à disposition: /info, /getservice, /enum et /ex !
Dans chaque élément, vous obtenez le lien vers la documentation (avec la recherche demandé).





Pour la commande /info -> /info [argument]

Concernant les arguments -> instance, player, mouse, coregui.

Exemple: /info player
Ce qui nous donne:

Sachez que ici vous ne voyez pas tout, il y a d'autres éléments donné!





Pour la commande /enum -> /enum [argument]

Pour les aruments vous avez -> cameramode, cameratype, keycode et material !

Exemple: /enum keycode
Ce qui nous donne:





Pour la commande /getservice -> /getservice [argument]

Pour les arguments vous avez -> workspace, team, userinputservice, contextactionservice, startergui, serverstorage, serverscriptservice, replicatedstorage, replicatedfirst, httpservice, players, runservice, tweenservice !

Exemple: /getservice runservice
Ce qui nous donne:





Pour la commande /ex -> /ex [argument: fichier]/[argument: élément]

Concernant les fichiers, ce sont les arguments que vous donnez dans la commande r!info ou r!getservice !
Ensuite, pour l'élément, vous pouvez regarder le fichier et ensuite prendre un élément de ce dernier, par exemple dans le RunService nous trouvons une fonction :Pause() !

Les éléments s'obtiennent grâce à la commande r!info ou r!getservice et en fonction de l'argument que vous donnerez, vous aurez la liste des éléments :

- propriétées
- fonctions
- évènements

Exemple : /ex userinputservice/inputbegan
Ce qui nous donne:

Notez que vous ne devez pas mettre paranthèses, deux points ou autre !

Petite note à avoir concernant les enums ! Vous devrez donner un 3ème argument OBLIGATOIREMENT ! Exemple : /ex enum/keycode/a Ce qui donnera un exemple sur les KeyCode avec la touche A.
